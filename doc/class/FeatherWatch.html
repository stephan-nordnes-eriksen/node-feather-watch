# (FeatherWatch) constructor(directories, callback, verbose = false, silence_exceptions = false)

Initialize the file system watcher.

Examples:

Make a new FeatherWatch object 

FeatherWatch::Watcher.new([File.join(Dir.home, 'Desktop')], lambda{|e| puts "got event with status: #{e[:status]} on file #{e[:file]}"})

Parameters:

  • directories ( Array ) — An array of directories to watch.
  • callback ( Function ) — A callback that will receive events. Recieved objects on form: {status: [:modified, :removed].sample, file: "path/to/file"}. ":modified" means everything except deleted.
  • verbose ( Boolean ) — Enable verbose mode. Warning: Can produce very much output. Default: false
